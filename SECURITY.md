# Security Policy

Enterprise Context Engine is a portfolio/demo project that uses synthetic data,
local mock users, and local-only API key simulation. Do not use demo API keys,
sample credentials, generated SQLite files, or evaluation artifacts as production
security controls.

For the implemented permission model, safe-abstention behavior, and current
limitations, see [docs/SECURITY.md](docs/SECURITY.md).

Please report suspected vulnerabilities privately to the project maintainer
instead of opening a public issue. Include the affected commit or version,
reproduction steps, observed impact, and any suggested fix.

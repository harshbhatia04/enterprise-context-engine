# Roadmap

## Near-Term

- Add real GitLab handbook crawler/downloader.
- Add Kubernetes, NIST, and SEC public-source ingestion.
- Add better real-source eval sets.
- Add screenshot assets for the dashboard and evaluation pages.

## Productionization

- OAuth/OIDC authentication.
- PostgreSQL metadata and log store.
- Object storage for source documents and artifacts.
- Background ingestion jobs.
- Observability and tracing.
- Secrets management and deployment hardening.

## Retrieval Quality

- Better rerankers.
- Query decomposition.
- Multi-hop retrieval.
- LLM-as-judge as optional secondary eval.
- Active learning from feedback.
- Larger regression suites for out-of-corpus and adversarial questions.
